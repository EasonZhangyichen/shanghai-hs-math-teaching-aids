# 正弦函数性质探究

资源 ID：`SH-HS-MATH-HJ-B2-C07-L02-A01`
版本：`0.1.0`
状态：`draft`
类型：HTML Applet 可运行原型

## 资源定位

本资源对应沪教版高中数学必修第二册第 7 章 7.1.2“正弦函数的性质”。它不重新讲正弦曲线来源，而是在同一张 `y = sin x` 图像上分步比较周期段、单调段、对称元素和关键点，让学生把性质从图像对象中读出来。

课堂主线是：

```text
图像上的当前点
-> 相隔 2pi 的同高点
-> 一个周期内的上升段与下降段
-> 对称轴与对称中心
-> 零点、最值点的一般形式
-> 性质归纳
```

## 为什么需要数字化

静态板书可以列出正弦函数性质，但很难让学生在一张图中反复比较“同高点”“相邻端点”“轴对称对象”“中心对称对象”。这个 Applet 只承担一个主要认知动作：比较。学生移动同一个观察点，教师逐层打开性质标注，帮助学生看到每条性质对应的图像证据。

## 课堂适用场景

- 新授归纳：在完成正弦图像后，系统归纳定义域、值域、周期性、奇偶性、单调性、零点和最值点。
- 易错澄清：区分对称轴 `x = pi/2 + kpi` 和对称中心 `(kpi, 0)`。
- 表达训练：把特殊点推广为带整数 `k` 的一般式。
- 课堂诊断前置：为后续 `SH-HS-MATH-HJ-B2-C07-L02-D01` 诊断任务提供图像回看入口。

## 资源边界

本资源包含：

- Applet metadata、教师脚本、学生活动任务和草稿审核记录。
- `src/index.html` 可运行原型。
- 周期比较、单调区间、对称元素、关键点和性质归纳的分步揭示。
- Applet SDK 基础消息：`applet:ready`、`applet:stateChanged`、`applet:interaction`、`applet:checkpoint`。

本资源暂不包含：

- 余弦函数、正切函数性质。
- 参数函数 `y = A sin(omega x + phi) + k` 的变换实验。
- 用导数证明单调性。
- 独立诊断题组与评分规则。

## 交互脚本概览

### 第 1 幕：只看图像和当前点

初始显示 `-2pi <= x <= 4pi` 上的正弦曲线、坐标轴和当前点。教师让学生先从图像读出值域和最基本的上下界。

### 第 2 幕：周期比较

打开周期比较点，显示当前点与相隔 `2pi` 的同高点。教师追问：横坐标变了什么，纵坐标为什么不变？

### 第 3 幕：单调区间比较

选择 `k = -1, 0, 1`，突出对应周期中的递增段和递减段。学生比较端点并写出：

```text
递增：[-pi/2 + 2kpi, pi/2 + 2kpi]
递减：[pi/2 + 2kpi, 3pi/2 + 2kpi]
```

### 第 4 幕：对称元素比较

显示对称轴和对称中心。教师要求学生说清：对称轴是竖直直线，对称中心是图像上的点，二者不是同一种对象。

### 第 5 幕：关键点一般化

显示零点、最大值点和最小值点。学生从多个周期中的同类点归纳：

```text
零点：x = kpi
最大值点：(pi/2 + 2kpi, 1)
最小值点：(3pi/2 + 2kpi, -1)
```

### 第 6 幕：性质归纳

打开性质归纳面板，教师带学生检查每条性质是否都能在图像上找到证据。

## 当前 HTML 原型能力

- 拖动图像或调整滑块控制当前 `x`，同步显示 `sin x` 和相位位置。
- 支持播放、暂停、重置和选定 `k` 周期段。
- 支持显示/隐藏周期比较、单调区间、对称元素、关键点和性质归纳。
- 支持关键点附近的即时提示，区分零点、最大值点、最小值点和当前单调方向。
- 当前仍是 `draft`，进入 `math_review` 前需要数学审校者复核一般式、边界端点和课堂节奏。

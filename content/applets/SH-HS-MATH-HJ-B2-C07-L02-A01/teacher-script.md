# 教师脚本：正弦函数性质探究

适用课时：必修第二册 7.1.2“正弦函数的性质”
建议时长：20 分钟
资源状态：`draft`

## 教学目标

1. 让学生从图像比较中解释正弦函数的周期性和值域。
2. 让学生用端点比较写出完整的单调区间一般式。
3. 让学生区分对称轴、对称中心和奇函数性质。
4. 让学生用 `k in Z` 表达零点、最大值点和最小值点。

## 课前准备

- 默认显示 `-2pi <= x <= 4pi` 上的正弦曲线。
- 初始隐藏周期比较点、单调段、对称元素、关键点表和性质归纳。
- 黑板预留三块区域：周期与值域、单调区间、对称与关键点。

## 课堂流程

### 1. 从图像读性质边界

时长：3 分钟

操作：

- 显示基础图像和当前点。
- 拖动当前点到几个位置，让学生观察纵坐标范围。

提问：

- 图像最高点和最低点分别是多少？
- 是否有某个横坐标不能代入 `sin x`？
- 只看这张图，哪些性质还不能靠一个点判断？

预期回答：

- 值域为 `[-1, 1]`，定义域为 `R`。
- 周期、单调性、对称性都需要比较多个点或多个区间。

### 2. 周期性：比较同高点

时长：4 分钟

操作：

- 打开“周期”层。
- 拖动当前点，观察当前点与相隔 `2pi` 的比较点。

核心追问：

- 两个点的横坐标相差多少？
- 它们的纵坐标有什么关系？
- 如果把 `2pi` 换成 `2kpi`，结论怎样写？

板书建议：

```text
sin(x + 2pi) = sin x
sin(x + 2kpi) = sin x, k in Z
最小正周期：2pi
```

即时纠偏：

- 如果学生只说“周期是 2pi”，要求补全函数值相等的表达式。
- 如果学生把 `kpi` 写成周期步长，回到相隔 `pi` 的点比较正负号。

### 3. 单调性：比较端点和方向

时长：5 分钟

操作：

- 打开“单调”层。
- 依次选择 `k = -1, 0, 1`，让学生看上升段和下降段如何平移。

提问链：

- 当 `k = 0` 时，上升段的两个端点是什么？
- 为什么从 `pi/2` 到 `3pi/2` 是下降？
- 切换到 `k = 1` 后，端点整体平移了多少？
- 一般式里的周期步长应该写 `kpi` 还是 `2kpi`？

板书建议：

```text
递增：[-pi/2 + 2kpi, pi/2 + 2kpi]
递减：[pi/2 + 2kpi, 3pi/2 + 2kpi]
k in Z
```

即时纠偏：

- 如果学生把递增段写成 `[0, pi/2]`，追问它是否覆盖完整上升过程。
- 如果遗漏 `2kpi`，切换周期段让学生比较端点平移。

### 4. 对称性：轴和中心不是同一种对象

时长：4 分钟

操作：

- 打开“对称”层。
- 同屏显示竖直对称轴和中心对称点。

核心追问：

- 竖直虚线表示什么对象？
- 图像上的方形点表示什么对象？
- 关于原点的对称说明了什么函数性质？

板书建议：

```text
对称轴：x = pi/2 + kpi
对称中心：(kpi, 0)
奇函数：sin(-x) = -sin x
```

即时纠偏：

- 如果学生说“对称轴是 `(kpi, 0)`”，要求区分直线和点。
- 如果只写原点一个中心，要求观察其他零点是否也能作为中心。

### 5. 零点和最值点：从特殊值到一般式

时长：3 分钟

操作：

- 打开“关键点”层。
- 让学生按颜色或标签找同类点。

提问：

- 零点之间的横坐标间隔是多少？
- 最大值点之间的横坐标间隔是多少？
- 最小值点可以写成 `-pi/2 + 2kpi` 吗？和 `3pi/2 + 2kpi` 有什么关系？

板书建议：

```text
零点：x = kpi
最大值点：(pi/2 + 2kpi, 1)
最小值点：(3pi/2 + 2kpi, -1)
k in Z
```

### 6. 收束检查

时长：1 分钟

操作：

- 打开“归纳”层。
- 逐条追问每条性质在图上对应的证据。

学生完成一句话：

```text
我判断正弦函数性质时，不能只背一个特殊周期，而要先确定比较对象，再写出带 k 的一般形式。
```

## 常见误区与处理

| 误区 | 表现 | 处理方式 |
| --- | --- | --- |
| 周期遗漏 | 只写 `sin(x + 2pi) = sin x`，不写 `2kpi` | 切换多个周期段，追问同样关系是否反复出现。 |
| 单调端点混乱 | 把递增写成 `[0, pi]` 或遗漏端点 | 打开单调层，让学生按上升和下降方向读端点。 |
| 对称轴与中心混淆 | 把点 `(kpi, 0)` 说成对称轴 | 同屏比较竖线和中心点，要求说出对象类型。 |
| 最值点一般式遗漏 | 只写 `pi/2` 或 `3pi/2` | 显示多个最大/最小点，比较横坐标差。 |

## 本节不展开

- 余弦函数和正切函数性质。
- 参数变换函数的图像变化。
- 导数证明单调性。

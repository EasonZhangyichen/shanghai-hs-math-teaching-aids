# 教师脚本：正弦余弦性质对照板

适用课时：必修第二册 7.2.2“余弦函数的性质”
建议时长：20 分钟
资源状态：`draft`

## 教学目标

1. 让学生比较正弦和余弦的性质差异，避免机械套用。
2. 让学生从余弦图像端点写出完整的单调区间一般式。
3. 让学生区分余弦函数的偶函数性质、对称轴和对称中心。
4. 让学生用 `k in Z` 表达余弦函数零点、最大值点和最小值点。

## 课前准备

- 默认显示 `-2pi <= x <= 4pi` 上的余弦曲线和正弦参照曲线。
- 初始隐藏周期比较、单调段、对称元素、关键点表和性质归纳。
- 黑板预留三块区域：正弦/余弦差异、单调与对称、关键点一般式。

## 课堂流程

### 1. 先识别余弦主图像

时长：3 分钟

操作：

- 重置到 `x = 0`。
- 保留正弦参照曲线，拖动同步游标到 `0`、`pi/2`、`pi`。

提问：

- 哪条曲线是余弦？判断依据是什么？
- `x = 0` 处余弦值是多少？正弦值是多少？
- 如果直接套正弦性质，最容易先套错什么？

预期回答：

- 余弦曲线从 `(0, 1)` 开始。
- 正弦是奇函数，余弦是偶函数；关键点顺序不同。

### 2. 周期性：同高点仍相隔 2pi

时长：3 分钟

操作：

- 打开“周期”层。
- 拖动当前点，观察当前余弦点与相隔 `2pi` 的比较点。

板书建议：

```text
cos(x + 2pi) = cos x
cos(x + 2kpi) = cos x, k in Z
最小正周期：2pi
```

即时纠偏：

- 如果学生写成 `kpi`，让他们比较相隔 `pi` 的点：函数值一般相反。

### 3. 单调性：从余弦端点读方向

时长：5 分钟

操作：

- 打开“单调”层。
- 切换 `k = -1, 0, 1`，让学生读每段端点。

提问链：

- 当 `k = 0` 时，余弦从 `x = 0` 到 `x = pi` 是上升还是下降？
- 余弦从 `x = -pi` 到 `x = 0` 呢？
- 切到 `k = 1` 后，端点整体平移了多少？
- 一般式里的周期步长为什么是 `2kpi`？

板书建议：

```text
递增：[-pi + 2kpi, 2kpi]
递减：[2kpi, pi + 2kpi]
k in Z
```

### 4. 对称性：偶函数、轴、中心分开说

时长：4 分钟

操作：

- 打开“对称”层。
- 停在正负同角位置，观察 `cos(-x) = cos x`。

提问链：

- 关于 y 轴的配对点说明了什么函数性质？
- 余弦图像的对称轴可以怎样写？
- 余弦图像有没有对称中心？它们在哪些点？

板书建议：

```text
偶函数：cos(-x) = cos x
对称轴：x = kpi
对称中心：(pi/2 + kpi, 0)
k in Z
```

即时纠偏：

- 如果学生说余弦是奇函数，回到 `x` 和 `-x` 的两个同高点。
- 如果把中心写成 `(kpi, 0)`，提示这些点是最大/最小点，不是中心。

### 5. 零点和最值点：不要把正弦序列搬过来

时长：4 分钟

操作：

- 打开“关键点”层。
- 让学生先读一个周期，再推广到所有周期。

板书建议：

```text
零点：x = pi/2 + kpi
最大值点：(2kpi, 1)
最小值点：(pi + 2kpi, -1)
k in Z
```

提问：

- 余弦零点之间的间隔是多少？
- 最大值点之间的间隔是多少？
- 为什么最大值点不能写成 `(pi/2 + 2kpi, 1)`？

### 6. 收束检查

时长：1 分钟

操作：

- 打开“归纳”层。
- 逐条追问每条性质在图上的证据。

学生完成一句话：

```text
我判断余弦函数性质时，先看余弦图像证据，再写出带 k 的一般形式，不能直接套正弦函数性质。
```

## 常见误区与处理

| 误区 | 表现 | 处理方式 |
| --- | --- | --- |
| 正弦性质迁移 | 把余弦写成奇函数或套正弦单调区间 | 保留正弦参照，比较 `x = 0` 附近的关键点顺序。 |
| 单调方向写反 | 写成 `[0, pi]` 递增 | 打开单调层，从 `(0, 1)` 到 `(pi, -1)` 读方向。 |
| 对称轴与中心混淆 | 把 `(kpi, 0)` 写成中心 | 指出 `(kpi, 0)` 不是余弦图像上的零点，回到中心对称点。 |
| 关键点一般式遗漏 | 只写 `0`、`pi/2`、`pi` | 显示多个周期，比较同类点横坐标差。 |

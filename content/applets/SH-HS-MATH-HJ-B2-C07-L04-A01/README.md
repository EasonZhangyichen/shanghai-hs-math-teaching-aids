# 正弦余弦性质对照板

资源 ID：`SH-HS-MATH-HJ-B2-C07-L04-A01`
版本：`0.1.0`
状态：`draft`
类型：HTML Applet 可运行原型

## 资源定位

本资源对应沪教版高中数学必修第二册第 7 章 7.2.2“余弦函数的性质”。它承接 7.2.1 的余弦图像生成和正弦/余弦相位对照，把课堂焦点转到性质归纳：定义域、值域、周期性、奇偶性、单调区间、对称元素、零点和最值点。

课堂主线是：

```text
正弦参照
-> 同一 x 下比较 sin x 与 cos x
-> 观察余弦关键点顺序
-> 分层显示周期、单调、对称和关键点
-> 写出 y = cos x 的一般形式
```

## 为什么需要数字化

余弦性质容易被学生“从正弦平移后背下来”，但真正出错的位置往往在一般式：单调区间端点错位、奇偶性套错、零点和最值点漏写周期。这个 Applet 只承担一个主要认知动作：比较。教师可以让正弦作为参照曲线保留或隐藏，并逐层打开余弦性质对象，让学生先看图像证据，再写带 `k in Z` 的结论。

## 课堂适用场景

- 学生刚学完余弦函数图像，需要系统归纳余弦性质。
- 学生把正弦函数的奇函数、单调区间、对称轴或关键点顺序套用到余弦函数。
- 教师需要在一张图中控制周期、单调、对称和关键点的揭示节奏。
- 后续进入三角函数参数图像前，用作正弦/余弦基本性质复盘底图。

## 资源边界

本资源包含：

- Applet metadata、教师脚本、学生活动任务和草稿审核记录。
- `src/index.html` 可运行原型。
- 正弦/余弦同屏曲线、同步游标、周期比较、单调段、对称元素、关键点表和性质归纳。
- Applet SDK 基础消息：`applet:ready`、`applet:stateChanged`、`applet:interaction`、`applet:checkpoint`。

本资源暂不包含：

- 余弦函数图像的单位圆生成过程。
- 参数函数图像变换实验。
- 正切函数性质。
- 诊断题自动批改。

## 当前 HTML 原型能力

- 拖动图像同步游标、调整 `x` 滑块、播放动画三种方式更新同一状态。
- 支持 `k = -1, 0, 1` 周期段切换，联动单调区间和性质表。
- 支持显示/隐藏正弦参照、周期比较、单调区间、对称元素、关键点和性质归纳。
- 支持关键角跳转，便于课堂停顿追问。
- 当前仍是 `draft`，进入 `math_review` 前需要数学审校者复核单调区间、对称中心和关键点一般式。

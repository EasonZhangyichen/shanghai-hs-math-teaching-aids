# 学生活动：从单位圆生成正弦曲线

适用资源：`SH-HS-MATH-HJ-B2-C07-L01-A01`  
完成方式：课堂观察记录、小组讨论、个人短答

## 活动目标

- 看懂 `sin theta` 是单位圆上点 P 的纵坐标。
- 说明正弦图像上点 Q 的坐标为什么是 `(theta, sin theta)`。
- 用单位圆解释正弦曲线的关键点和周期重复。

## 任务一：先填关键值

观察单位圆上动点 P 的位置，先不要看完整曲线。完成表格。

| `theta` | P 在单位圆上的位置 | `sin theta` | Q 在图像上的坐标 |
| --- | --- | --- | --- |
| `0` | `(1, 0)` |  |  |
| `pi/2` | `(0, 1)` |  |  |
| `pi` | `(-1, 0)` |  |  |
| `3pi/2` | `(0, -1)` |  |  |
| `2pi` | `(1, 0)` |  |  |

检查问题：

1. Q 的横坐标记录的是 P 的横坐标，还是角 `theta`？
2. Q 的纵坐标和 P 的哪个坐标相同？

## 任务二：预测图像点的运动

在教师显示轨迹前，先根据单位圆预测 Q 的运动。

1. 当 `theta` 从 `0` 增大到 `pi/2` 时，P 的纵坐标怎样变化？Q 应向上还是向下？
2. 当 `theta` 从 `pi/2` 增大到 `pi` 时，P 仍在上半圆，为什么 Q 的纵坐标会变小？
3. 当 `theta` 从 `pi` 增大到 `3pi/2` 时，`sin theta` 的符号是什么？
4. 当 `theta` 到达 `2pi` 时，P 和 Q 分别回到什么状态？

## 任务三：解释一句话

用自己的话补全下面的说明。

```text
单位圆上点 P 的坐标是 (cos theta, sin theta)。
正弦函数图像上的点 Q 不是 P 本身，
而是把 __________ 作为横坐标，把 __________ 作为纵坐标，
所以 Q = (__________, __________)。
```

## 任务四：周期理由

观察 `theta` 和 `theta + 2pi`。

1. 它们的终边是否相同？
2. 它们在单位圆上对应的点 P 是否相同？
3. 为什么可以得到 `sin(theta + 2pi) = sin theta`？

请用“终边”“单位圆上的点”“纵坐标”三个词写一句完整解释。

## 任务五：快速诊断

判断下列说法是否正确，并说明理由。

1. 正弦函数图像上的点可以写成 `(cos theta, sin theta)`。
2. 当 P 在单位圆下半部分时，`sin theta` 小于 0。
3. 因为转过 `2pi` 后 P 回到同一点，所以正弦曲线会重复。

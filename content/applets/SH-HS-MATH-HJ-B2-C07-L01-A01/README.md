# 单位圆到正弦曲线

资源 ID：`SH-HS-MATH-HJ-B2-C07-L01-A01`  
版本：`0.1.0`  
状态：`draft`  
类型：HTML Applet 规格包，暂不包含 HTML 实现

## 资源定位

本资源对应沪教版高中数学必修第二册第 7 章 7.1.1“正弦函数的图像”。它要解决的核心问题不是“正弦曲线长什么样”，而是让学生看见正弦曲线如何由单位圆上的旋转运动生成。

课堂中的主线是：

```text
单位圆动点 P
-> P 的纵坐标 sin theta
-> 图像点 Q = (theta, sin theta)
-> Q 的轨迹形成 y = sin x
-> 终边相同角带来 2pi 周期重复
```

## 为什么需要数字化

静态板书可以画单位圆和正弦曲线，但很难同时保持三件事同步变化：转过的角、单位圆动点的纵坐标、函数图像上的点。这个 Applet 只承担一个认知动作：投影。学生拖动动点时，应持续看到同一个角度在两套坐标系统中的对应关系。

## 课堂适用场景

- 新授课导入：从第 6 章单位圆中的正弦定义过渡到第 7 章函数图像。
- 探究活动：学生先预测关键角处的函数值，再拖动验证。
- 概念澄清：纠正“图像横坐标是单位圆上点横坐标”的误解。
- 周期解释：用终边重合解释 `sin(x + 2pi) = sin x`。

## 交互脚本概览

### 第 1 幕：只看单位圆

初始只显示单位圆、坐标轴、动点 P、角 `theta`。教师拖动或播放动点，让学生回忆：`sin theta` 是单位圆上点 P 的纵坐标。

### 第 2 幕：显示投影

打开投影线，显示 P 的纵坐标如何成为当前的 `sin theta`。此时不要急于显示完整曲线，先让学生在 `0`、`pi/2`、`pi`、`3pi/2`、`2pi` 处说出符号和数值。

### 第 3 幕：生成图像点

在右侧坐标系显示点 `Q = (theta, sin theta)`。教师追问：Q 的横坐标为什么是 `theta`，不是 P 的横坐标 `cos theta`？

### 第 4 幕：留下轨迹

拖动 P 一周，保留 Q 的轨迹。学生观察五个关键点连成的平滑曲线，并用“角度变化”和“纵坐标变化”解释上升、下降、过零、到达最值。

### 第 5 幕：周期延拓

切换到周期延拓视图，同时显示 `theta` 与 `theta + 2pi` 的终边重合，让学生从单位圆来源解释曲线重复。

## 后续 HTML 实现验收点

- 拖动 P、调整 `theta` 滑块、播放动画三种方式必须更新同一份状态。
- 单位圆动点、投影线、图像点、轨迹和关键值表必须同步。
- 初始界面不得直接展示完整结论，应支持教师分步揭示。
- 必须有重置、暂停、显示/隐藏轨迹、显示/隐藏关键点等课堂控制。
- 大屏投影下关键点标签和公式可读，触控拖动不应误触。
- 反馈不只显示对错，要引导学生回到 `P(cos theta, sin theta)` 与 `Q(theta, sin theta)` 的对应。

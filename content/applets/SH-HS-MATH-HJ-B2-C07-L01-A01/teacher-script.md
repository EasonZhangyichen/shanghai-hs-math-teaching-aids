# 教师脚本：单位圆到正弦曲线

适用课时：必修第二册 7.1.1“正弦函数的图像”
建议时长：18 分钟
资源状态：`draft`

## 教学目标

1. 让学生把 `sin theta` 理解为单位圆上动点 P 的纵坐标。
2. 让学生说清正弦图像上的点为什么是 `Q = (theta, sin theta)`。
3. 让学生从单位圆一周旋转生成 `0 <= theta <= 2pi` 上的正弦曲线。
4. 让学生用终边相同角解释正弦曲线的周期重复。

## 课前准备

- 默认角度范围设为 `0 <= theta <= 2pi`。
- 初始只显示单位圆、坐标轴、动点 P 和角 `theta`。
- 暂时隐藏完整轨迹、关键值表和周期延拓。
- 黑板预留两块区域：左侧写单位圆关系，右侧写函数图像关系。

## 课堂流程

### 1. 复现旧知：正弦值从哪里来

时长：3 分钟

操作：

- 将 P 放在第一象限，显示 `P(cos theta, sin theta)`。
- 拖动 P 到几个不同象限，但暂不显示图像轨迹。

提问：

- 当 P 在单位圆上运动时，哪个坐标表示 `sin theta`？
- P 到达上端、左端、下端、右端时，正弦值分别是多少？

预期回答：

- `sin theta` 是 P 的纵坐标。
- 关键值依次为 `0, 1, 0, -1, 0`。

即时纠偏：

- 如果学生说“P 的横坐标是正弦值”，回到坐标标注，要求先读出 P 的坐标再判断。

### 2. 建立新表征：从 P 到 Q

时长：5 分钟

操作：

- 打开投影线，突出 P 的纵坐标。
- 在右侧坐标系显示图像点 `Q = (theta, sin theta)`，但仍隐藏轨迹。
- 依次停在 `0`、`pi/2`、`pi`、`3pi/2`、`2pi`。

核心追问：

- 图像点 Q 的横坐标为什么写 `theta`？
- Q 的纵坐标和 P 的哪个量相同？
- P 的横坐标 `cos theta` 在这个图像中扮演什么角色？

预期回答：

- 因为函数 `y = sin x` 的自变量是角的弧度量。
- Q 的纵坐标等于 P 的纵坐标，也就是 `sin theta`。
- P 的横坐标不是正弦图像的横坐标，它只是单位圆表征中的横坐标。

板书建议：

```text
单位圆：P = (cos theta, sin theta)
函数图像：Q = (theta, sin theta)
横坐标变了：从平面位置 -> 角的弧度量
纵坐标相同：都是 sin theta
```

### 3. 先预测再生成：五个关键点连成曲线

时长：5 分钟

操作：

- 隐藏轨迹，只显示当前 Q。
- 让学生先预测下一关键角处 Q 的位置。
- 拖动 P 一周，打开“保留轨迹”。

提问链：

- 从 `0` 到 `pi/2`，P 的纵坐标如何变化？图像上的 Q 应该向哪里走？
- 从 `pi/2` 到 `pi`，P 仍在上半圆，为什么图像反而下降？
- 从 `pi` 到 `3pi/2`，正弦值的符号发生了什么变化？
- 到 `2pi` 时，P 回到哪里？Q 的纵坐标回到多少？

预期归纳：

- 正弦曲线不是凭空画出的波浪线，而是 Q 随角度变化留下的轨迹。
- 上升、下降、过零、到最值都来自单位圆纵坐标的变化。

### 4. 解释周期：为什么曲线重复

时长：3 分钟

操作：

- 切换到周期延拓视图。
- 同时显示 `theta` 和 `theta + 2pi` 对应的终边。

提问：

- `theta` 和 `theta + 2pi` 的终边有什么关系？
- 它们在单位圆上对应的 P 是否相同？
- 因此 `sin(theta + 2pi)` 和 `sin theta` 有什么关系？

板书建议：

```text
终边相同 -> 单位圆上的点相同 -> 纵坐标相同
sin(theta + 2pi) = sin theta
```

### 5. 收束表达：一句话说明来源

时长：2 分钟

学生口头或书面完成：

```text
当角 theta 变化时，单位圆上点 P 的纵坐标是 sin theta；
把 theta 作为横坐标、sin theta 作为纵坐标，就得到正弦函数图像。
```

## 常见误区与处理

| 误区 | 表现 | 处理方式 |
| --- | --- | --- |
| 把 Q 的横坐标当成 P 的横坐标 | 认为图像点是 `(cos theta, sin theta)` | 同屏对比 P 与 Q，追问函数图像的自变量是什么。 |
| 会背关键点但不会解释 | 能说出五点，不能说明来源 | 隐藏轨迹，只用单位圆预测下一个 Q 的位置。 |
| 把周期当成记忆结论 | 直接说周期是 `2pi` | 显示 `theta` 与 `theta + 2pi` 终边重合，要求从点相同推出纵坐标相同。 |
| 弧度刻度不敏感 | 把 `pi/2`、`pi` 看成普通标签 | 在单位圆上同步显示角弧，强调横轴记录的是角的弧度量。 |

## 本节不展开

- 参数变化 `A`、`omega`、`phi` 对图像的影响。

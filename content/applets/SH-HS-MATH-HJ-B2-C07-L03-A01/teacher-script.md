# 教师脚本：单位圆到余弦曲线

适用课时：必修第二册 7.2.1“余弦函数的图像”
建议时长：18 分钟
资源状态：`draft`

## 教学目标

1. 让学生从单位圆动点的横坐标解释 `cos theta`。
2. 让学生建立图像点 `Q = (theta, cos theta)` 的坐标含义。
3. 让学生按角度顺序说出余弦图像五个关键点。
4. 让学生把余弦曲线与正弦参照联系起来，但不把余弦来源简化为平移记忆。

## 课前准备

- 初始显示单位圆、坐标轴、动点 P、角 `theta` 和右侧坐标系。
- 默认打开横向投影线和余弦轨迹；正弦参照与周期延拓先隐藏。
- 黑板预留三块区域：单位圆坐标来源、关键点表、正弦与余弦关系。

## 课堂流程

### 1. 回到单位圆中的余弦定义

时长：3 分钟

操作：

- 重置到 `theta = 0`。
- 拖动点 P 在第一象限内缓慢移动。
- 只保留单位圆和横向投影，暂不强调完整曲线。

提问：

- 点 P 的坐标可以怎样表示？
- `cos theta` 对应 P 的哪个坐标？
- 如果 P 向上转动，P 的横坐标在变大还是变小？

预期回答：

- `P = (cos theta, sin theta)`。
- `cos theta` 是 P 的横坐标。
- 从 `0` 到 `pi/2`，横坐标从 `1` 减小到 `0`。

### 2. 把横坐标投影到函数图像

时长：4 分钟

操作：

- 显示右侧图像点 `Q = (theta, cos theta)`。
- 调整滑块到 `0`、`pi/2`、`pi`、`3pi/2`、`2pi`。

核心追问：

- Q 的横坐标为什么不是 P 的横坐标？
- Q 的纵坐标为什么等于 P 的横坐标？
- 五个关键角处的余弦值顺序是什么？

板书建议：

```text
P = (cos theta, sin theta)
Q = (theta, cos theta)

theta:     0   pi/2   pi   3pi/2   2pi
cos theta: 1     0    -1      0      1
```

即时纠偏：

- 如果学生把图像横坐标说成 `cos theta`，要求回到函数图像定义：自变量是角的弧度量。
- 如果学生套用正弦关键点顺序，回到 P 的横坐标读值。

### 3. 生成余弦曲线

时长：4 分钟

操作：

- 打开轨迹，播放或拖动 P 一周。
- 在关键点附近暂停，让学生补齐图像点。

提问链：

- 为什么余弦图像从最大值 `1` 开始？
- 从 `0` 到 `pi`，图像为什么一直下降？
- 到 `2pi` 时，P 回到哪里？图像点对应什么值？

板书建议：

```text
y = cos x 的一周关键点：
(0, 1), (pi/2, 0), (pi, -1), (3pi/2, 0), (2pi, 1)
```

### 4. 打开正弦参照，比较坐标来源

时长：4 分钟

操作：

- 打开“正弦参照”。
- 让学生观察同一 `theta` 下的余弦点和正弦点。

核心追问：

- 同一个 P，余弦取的是哪个坐标，正弦取的是哪个坐标？
- 余弦曲线与正弦曲线看起来有什么平移关系？
- 这说明我们可以怎样理解 `cos x = sin(x + pi/2)`？

板书建议：

```text
同一 theta：
cos theta <- P 的横坐标
sin theta <- P 的纵坐标

关系：cos x = sin(x + pi/2)
来源：仍然是单位圆横坐标投影
```

即时纠偏：

- 如果学生只说“余弦就是正弦右移/左移”，要求先指出 P 的横坐标来源，再谈曲线关系。
- 如果学生把平移方向说反，用 `x = 0` 处的函数值快速检查。

### 5. 周期重复来自终边重合

时长：2 分钟

操作：

- 打开周期延拓视图。
- 同时展示 `theta` 与 `theta + 2pi` 的终边重合。

提问：

- 为什么两个角对应相同的 P？
- 为什么它们的 `cos` 值相同？
- 这能怎样解释余弦图像的周期重复？

### 6. 收束检查

时长：1 分钟

学生完成一句话：

```text
余弦函数图像上的点 (x, cos x) 来自单位圆中角 x 对应点的________坐标。
```

预期填空：横。

## 常见误区与处理

| 误区 | 表现 | 处理方式 |
| --- | --- | --- |
| 横纵坐标混淆 | 把 `cos theta` 说成 P 的纵坐标 | 关闭图像，只看单位圆和横向投影。 |
| 图像横坐标混淆 | 把 Q 的横坐标写成 `cos theta` | 回到函数图像点 `Q = (自变量, 函数值)`。 |
| 关键点顺序混用 | 把余弦关键值写成 `0, 1, 0, -1, 0` | 逐个停在单位圆关键位置读横坐标。 |
| 相位关系死记 | 只说“余弦是正弦平移” | 先追问单位圆坐标来源，再打开正弦参照。 |

## 本节不展开

- 余弦函数单调性、奇偶性和对称性的系统归纳。
- 正弦与余弦性质的完整比较。
- 参数函数图像变换。

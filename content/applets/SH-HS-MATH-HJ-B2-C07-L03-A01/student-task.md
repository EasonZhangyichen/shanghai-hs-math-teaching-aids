# 学生活动：从单位圆生成余弦曲线

适用资源：`SH-HS-MATH-HJ-B2-C07-L03-A01`
完成方式：课堂观察记录、小组讨论、个人短答

## 活动目标

- 从单位圆动点的横坐标说明 `cos theta`。
- 写出余弦图像点 `Q = (theta, cos theta)` 的坐标含义。
- 识别余弦图像一周内的关键点顺序。
- 比较同一角度下 `sin theta` 与 `cos theta` 的坐标来源。

## 任务五：快速诊断

判断下列说法是否正确，并说明理由。

1. `cos theta` 是单位圆动点 P 的纵坐标。
2. 余弦图像上的点可以写成 `(theta, cos theta)`。
3. `y = cos x` 在 `x = 0` 处的函数值是 `0`。
4. `theta` 与 `theta + 2pi` 对应单位圆上同一个点，所以余弦值相同。
5. 余弦曲线与正弦曲线有相位关系，但余弦函数值的来源仍是单位圆横坐标。

# 单位圆到余弦曲线

资源 ID：`SH-HS-MATH-HJ-B2-C07-L03-A01`
版本：`0.1.0`
状态：`draft`
类型：HTML Applet 可运行原型

## 资源定位

本资源对应沪教版高中数学必修第二册第 7 章 7.2.1“余弦函数的图像”。它要解决的核心问题不是让学生记住“余弦曲线像正弦曲线平移”，而是让学生先看见余弦函数值来自单位圆上动点的横坐标。

课堂主线是：

```text
单位圆动点 P
-> P 的横坐标 cos theta
-> 图像点 Q = (theta, cos theta)
-> Q 的轨迹形成 y = cos x
-> 与 sin theta 参照比较，解释相位关系
```

## 为什么需要数字化

静态板书可以画出余弦曲线，也可以写出 `cos x = sin(x + pi/2)`，但很难在同一时刻同步展示角、单位圆横坐标、图像点和正弦参照。这个 Applet 只承担一个主要认知动作：投影。学生拖动单位圆动点时，应持续看到同一个角度在单位圆和函数图像中的对应关系。

## 课堂适用场景

- 新授课导入：从单位圆中的余弦定义过渡到余弦函数图像。
- 探究活动：学生先预测关键角处的余弦值，再拖动验证轨迹。
- 概念澄清：纠正“余弦图像只是正弦曲线平移记忆”的单一路径。
- 对照引导：打开正弦参照，说明同一角度下横坐标投影和纵坐标投影的区别。

## 交互脚本概览

### 第 1 幕：只看单位圆

初始显示单位圆、坐标轴、动点 P 和角 `theta`。教师拖动或播放动点，让学生回忆：`cos theta` 是单位圆上点 P 的横坐标。

### 第 2 幕：显示横向投影

打开投影线，显示 P 的横坐标如何成为当前的 `cos theta`。此时先停在 `0`、`pi/2`、`pi`、`3pi/2`、`2pi`，让学生说出余弦值顺序 `1, 0, -1, 0, 1`。

### 第 3 幕：生成图像点

在右侧坐标系显示点 `Q = (theta, cos theta)`。教师追问：Q 的横坐标为什么是 `theta`，而纵坐标为什么是 P 的横坐标？

### 第 4 幕：留下轨迹

拖动 P 一周，保留 Q 的轨迹。学生观察五个关键点连成余弦曲线，并用“角度变化”和“横坐标变化”解释从最大值开始、过零、到最小值、再回到最大值。

### 第 5 幕：打开正弦参照

打开弱化的正弦曲线和同角正弦点，让学生比较 `cos theta` 与 `sin theta` 的坐标来源，并说明为什么 `cos x = sin(x + pi/2)` 是关系，不是余弦图像的唯一来源。

### 第 6 幕：周期延拓

切换到周期延拓视图，同时显示 `theta` 与 `theta + 2pi` 的终边重合，让学生从单位圆来源解释曲线重复。

## 当前 HTML 原型能力

- 拖动 P、调整 `theta` 滑块、播放动画三种方式更新同一份状态。
- 单位圆动点、横向投影线、图像点、轨迹、关键值表和正弦参照同步。
- 支持“单位圆 -> 横向投影 -> 图像点 -> 轨迹 -> 正弦参照 -> 周期延拓”的分步揭示。
- 支持重置、暂停、显示/隐藏投影线、轨迹、关键点、正弦参照和周期延拓。
- 按 Applet SDK 发送 `applet:ready`、`applet:stateChanged`、`applet:interaction` 和 `applet:checkpoint`。
- 当前仍是 `draft`，进入 `math_review` 前需要数学审校者复核符号、边界和课堂节奏。

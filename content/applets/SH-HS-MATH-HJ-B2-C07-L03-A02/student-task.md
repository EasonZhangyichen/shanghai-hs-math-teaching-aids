# 学生活动：比较正弦与余弦的相位

适用资源：`SH-HS-MATH-HJ-B2-C07-L03-A02`
完成方式：课堂观察记录、小组讨论、个人短答

## 活动目标

- 区分同一角 `x` 下 `sin x` 与 `cos x` 的坐标来源。
- 比较正弦、余弦在关键角处的值序列。
- 从图像上验证 `cos x = sin(x + pi/2)`。
- 用特殊值检查相位方向是否正确。

## 任务一：同一角下看单位圆

拖动单位圆上的点 P，填写表格。

| 角 `x` | P 的横坐标 | P 的纵坐标 | `cos x` | `sin x` |
| --- | --- | --- | --- | --- |
| `0` |  |  |  |  |
| `pi/2` |  |  |  |  |
| `pi` |  |  |  |  |
| `3pi/2` |  |  |  |  |
| `2pi` |  |  |  |  |

回答：

1. `sin x` 对应 P 的哪一个坐标？
2. `cos x` 对应 P 的哪一个坐标？
3. 同一个角下，为什么 `sin x` 和 `cos x` 通常不相等？

## 任务三：验证相位关系

打开“相位参照”，观察正弦曲线上 `x + pi/2` 处的点。

## 任务四：快速诊断

判断下列说法是否正确，并说明理由。

1. 同一角 `x` 下，`sin x` 取单位圆动点的横坐标。
2. 同一角 `x` 下，`cos x` 取单位圆动点的横坐标。
3. `cos x = sin(x - pi/2)`。
4. `cos x = sin(x + pi/2)` 可以用 `x = 0` 处的值验证方向。
5. 即使知道余弦曲线和正弦曲线有相位关系，也仍要能说出余弦来自单位圆横坐标。

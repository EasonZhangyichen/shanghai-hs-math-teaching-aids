# 教师脚本：正弦与余弦相位对照

适用课时：必修第二册 7.2.1“余弦函数的图像”
建议时长：16 分钟
资源状态：`draft`

## 教学目标

1. 让学生在同一角 `x` 下区分 `sin x` 与 `cos x` 的单位圆坐标来源。
2. 让学生用同步游标读出正弦和余弦的关键值序列。
3. 让学生从图像上解释 `cos x = sin(x + pi/2)`。
4. 让学生能用 `x = 0` 的值检查相位平移方向。

## 课前准备

- 初始显示单位圆、两条曲线和同步游标。
- 默认显示单位圆坐标投影和关键角表。
- 相位参照点与结论面板先隐藏，等学生完成同角比较后再打开。
- 黑板预留两块区域：同角坐标来源、相位关系检查。

## 课堂流程

### 1. 同一角下先看坐标来源

时长：3 分钟

操作：

- 重置到 `x = 0`。
- 缓慢拖动单位圆动点到第一象限。
- 保持坐标投影显示。

提问：

- 点 P 的坐标如何表示？
- 同一角 `x` 下，`sin x` 取 P 的哪个坐标？
- 同一角 `x` 下，`cos x` 取 P 的哪个坐标？

预期回答：

- `P = (cos x, sin x)`。
- `sin x` 取纵坐标，`cos x` 取横坐标。

### 2. 用同步游标比较两条曲线

时长：4 分钟

操作：

- 依次跳转到 `0`、`pi/2`、`pi`、`3pi/2`、`2pi`。
- 每到一个关键角，先让学生口答，再显示表格中的当前行。

板书建议：

```text
x:      0   pi/2   pi   3pi/2   2pi
sin x:  0     1     0     -1      0
cos x:  1     0    -1      0      1
```

即时纠偏：

- 如果学生把余弦值序列说成正弦序列，回到单位圆读横坐标。
- 如果学生只看曲线形状，要求先说“同一个 x 的两条竖直读数”。

### 3. 显示相位参照点

时长：4 分钟

操作：

- 打开“相位参照”。
- 停在 `x = 0`，显示正弦曲线上 `x + pi/2` 的点。
- 再移动到 `pi/2`、`pi` 做两次验证。

核心追问：

- 当 `x = 0` 时，`cos 0` 等于多少？
- 正弦曲线上哪个位置的值也是 `1`？
- 这说明 `cos x` 对应正弦曲线向哪个方向取值？

板书建议：

```text
cos x = sin(x + pi/2)

检查 x = 0：
cos 0 = 1
sin(0 + pi/2) = 1
```

### 4. 区分“相位关系”和“函数来源”

时长：3 分钟

操作：

- 打开结论面板。
- 同时保留单位圆和相位参照。

提问链：

- 说 `cos x = sin(x + pi/2)` 时，我们是在比较哪两条曲线？
- 说 `cos x` 的定义来源时，我们应该回到单位圆的哪个坐标？
- 为什么不能只说“余弦就是正弦平移”？

### 5. 收束检查

时长：2 分钟

学生完成两句话：

```text
同一角 x 下，sin x 来自单位圆点的________坐标，cos x 来自________坐标。
从图像关系看，cos x = sin(________)。
```

预期填空：纵；横；`x + pi/2`。

## 常见误区与处理

| 误区 | 表现 | 处理方式 |
| --- | --- | --- |
| 同角值混用 | 把 `sin x` 和 `cos x` 的读值互换 | 关闭相位参照，只看单位圆坐标投影。 |
| 关键点顺序混用 | 把余弦序列写成 `0, 1, 0, -1, 0` | 用关键角跳转逐个读横坐标。 |
| 相位方向说反 | 写成 `cos x = sin(x - pi/2)` | 用 `x = 0` 检查：右侧变成 `sin(-pi/2) = -1`。 |
| 平移替代来源 | 只说余弦是正弦平移 | 要求同时说出单位圆横坐标来源。 |

## 本节不展开

- 余弦函数单调性、奇偶性和对称性的系统归纳。
- 参数函数图像变换。
- 诱导公式的完整证明。

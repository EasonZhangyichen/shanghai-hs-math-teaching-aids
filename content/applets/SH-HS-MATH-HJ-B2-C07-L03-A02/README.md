# 正弦与余弦相位对照

资源 ID：`SH-HS-MATH-HJ-B2-C07-L03-A02`
版本：`0.1.0`
状态：`draft`
类型：HTML Applet 可运行原型

## 资源定位

本资源对应沪教版高中数学必修第二册第 7 章 7.2.1“余弦函数的图像”。它衔接 `SH-HS-MATH-HJ-B2-C07-L03-A01` 的单位圆投影主线，把课堂焦点从“余弦图像如何生成”推进到“正弦与余弦在同一角度下怎样比较，以及余弦曲线相对正弦曲线如何错开”。

课堂主线是：

```text
同一角 x
-> P = (cos x, sin x)
-> 同一 x 下读 y = sin x 与 y = cos x
-> 比较关键角值序列
-> 用 sin(x + pi/2) 验证 cos x 的相位位置
```

## 为什么需要数字化

板书能列出 `cos x = sin(x + pi/2)`，但学生常把它记成单纯平移结论，甚至把方向说反。这个 Applet 只承担一个主要认知动作：比较。拖动同一个角时，单位圆坐标、正弦点、余弦点和相位参照点同步变化，帮助学生把“同角比较”和“相位错开”分清。

## 课堂适用场景

- 余弦图像生成之后，用于澄清正弦、余弦的坐标来源。
- 学生混用正弦和余弦关键点顺序时，用同步游标逐点读值。
- 讲 `cos x = sin(x + pi/2)` 前，用 `x = 0` 的函数值检查平移方向。
- 后续进入余弦性质或参数图像前，建立正余弦图像对照底图。

## 资源边界

本资源包含：

- Applet metadata、教师脚本、学生活动任务和草稿审核记录。
- `src/index.html` 可运行原型。

- 诱导公式的完整代数证明。
- 诊断题自动批改。

## 当前 HTML 原型能力

- 拖动单位圆动点、调整 `x` 滑块、播放动画三种方式更新同一状态。
- 同步显示 `sin x`、`cos x`、`sin(x + pi/2)` 的图像位置。
- 支持显示/隐藏坐标投影、相位参照、关键角表和结论。
- 支持关键角跳转，便于课堂停顿追问。
- 当前仍是 `draft`，进入 `math_review` 前需要数学审校者复核相位方向、关键值和课堂语言。

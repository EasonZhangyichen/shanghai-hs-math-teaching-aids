# 教师脚本：三角函数参数变化实验室

适用课时：必修第二册 7.3“函数 y = A sin(omega x + phi) 的图像”
建议时长：25 分钟
资源状态：`draft`

## 教学目标

1. 让学生分离说明 `A`、`omega`、`phi` 对图像的作用。
2. 让学生理解周期 `T = 2pi / omega`，避免把 `omega` 与周期同向理解。
3. 让学生解释相位平移量 `-phi / omega`，而不是只看 `phi` 的符号。
4. 让学生从振幅、周期、关键点位置反推参数。

## 课前准备

- 初始参数设为 `A = 1`、`omega = 1`、`phi = 0`。
- 默认显示基本曲线、参数函数曲线、中间曲线和关键点。
- 黑板预留三块区域：振幅、周期、相位平移。

## 课堂流程

### 1. 建立基本参照

时长：3 分钟

操作：

- 显示 `y = sin x` 和当前参数函数重合的初始状态。
- 拖动观察点，让学生确认当前图像仍是基本正弦曲线。

提问：

- 当前振幅、周期和平移量分别是多少？
- 如果只看一张图，哪些参数可能互相干扰？

### 2. 单独改变 A：振幅与翻折

时长：5 分钟

操作：

- 切换到“A”单参数模式。
- 让 `A` 从 `1` 变到 `2`，再变到 `-1`。

提问：

- 最高点和最低点发生了什么变化？
- `A < 0` 时，图像只是振幅变大吗？
- 振幅应写 `A` 还是 `|A|`？

板书建议：

```text
振幅：|A|
A < 0：关于 x 轴翻折，再按 |A| 纵向伸缩
```

### 3. 单独改变 omega：周期反向变化

时长：5 分钟

操作：

- 切换到“omega”单参数模式。
- 让 `omega` 从 `1` 增大到 `2`，再减小到 `0.5`。

提问链：

- `omega` 变大时，一个完整周期在 x 轴上占的长度变大还是变小？
- 如果周期是 `pi`，`omega` 应该是多少？
- 为什么周期公式是 `2pi / omega`？

板书建议：

```text
T = 2pi / omega  (omega > 0)
omega 越大，周期越短
```

### 4. 单独改变 phi：平移方向与除以 omega

时长：6 分钟

操作：

- 切换到“phi”单参数模式。
- 先保持 `omega = 1`，调节 `phi`。
- 再切换自由模式，把 `omega` 改为 `2`，比较相同 `phi` 下的平移量。

核心追问：

- `omega x + phi = 0` 时，零点在哪里？
- 图像最终平移量为什么是 `-phi / omega`？
- `phi = pi/2` 一定表示左移 `pi/2` 吗？

板书建议：

```text
omega x + phi = 0
x = -phi / omega
相位平移量：-phi / omega
```

### 5. 变换顺序比较

时长：4 分钟

操作：

- 切换“先横向伸缩后平移”和“先平移后横向伸缩”。
- 打开中间曲线，让学生观察每一步对应的表达式。

提问：

- 哪一步改变周期？
- 哪一步决定最终零点位置？
- 两种顺序都能到达同一条最终曲线吗？中间曲线为什么不同？

### 6. 从图像反推参数

时长：2 分钟

操作：

- 显示关键点。
- 给出当前图像，让学生按顺序说出振幅、周期、相位平移。

学生完成一句话：

```text
我反推参数时，先看 |A|，再由周期求 omega，最后用零点或最值点求 -phi / omega。
```

## 常见误区与处理

| 误区 | 表现 | 处理方式 |
| --- | --- | --- |
| 横向伸缩反向 | `omega` 变大却说周期变大 | 显示一个完整周期长度，回到 `T = 2pi / omega`。 |
| 平移方向混淆 | 把 `+phi` 说成右移 | 解方程 `omega x + phi = 0`，读出 `x = -phi / omega`。 |
| 遗漏除以 omega | 只写平移量 `-phi` | 把 `omega = 1` 与 `omega = 2` 的同一 `phi` 对照。 |
| 忽略 A 为负 | 只说振幅变为 A | 先取 `|A|`，再说明关于 x 轴翻折。 |
| 多参数干扰 | 不能判断图像变化来源 | 回到单参数模式逐个恢复默认值。 |

变换顺序

图层

参数归纳

  • A：振幅为 1，负值表示关于 x 轴翻折。
  • omega：周期 2π，omega 越大周期越短。
  • phi：平移量 0，必须除以 omega。

课堂收束

  • 先看 |A| 判断振幅和翻折。
  • 再用 T = 2π / omega 判断横向伸缩。
  • 最后用 x = -phi / omega 判断相位平移。

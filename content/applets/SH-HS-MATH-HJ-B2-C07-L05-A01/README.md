# 三角函数参数变化实验室

资源 ID：`SH-HS-MATH-HJ-B2-C07-L05-A01`
版本：`0.1.0`
状态：`draft`
类型：HTML Applet 可运行原型

## 资源定位

本资源对应沪教版高中数学必修第二册第 7 章 7.3“函数 `y = A sin(omega x + phi)` 的图像”。它不把参数结论直接列成表格，而是让教师和学生通过单参数锁定、变换顺序切换和关键点追踪，观察基本正弦曲线如何变成参数函数图像。

课堂主线是：

```text
y = sin x
-> 单独改变 A、omega、phi
-> 比较两种变换顺序
-> 追踪零点和最值点
-> 归纳振幅、周期、相位平移
-> 从图像反推参数
```

## 为什么需要数字化

静态板书可以说明 `A` 控制振幅、`omega` 控制周期、`phi` 控制相位，但学生常常在三个地方出错：

- 把 `omega` 变大理解成周期变大。
- 把 `phi` 的符号直接当成图像平移方向。
- 多参数同时变化时无法判断哪个参数造成哪种视觉变化。

这个 Applet 只承担一个主要认知动作：变换。教师先让学生锁定单个参数，再逐步打开中间曲线和关键点，避免学生在一开始就面对完整复合变换。

## 当前 HTML 原型能力

- 支持调节 `A`、`omega`、`phi`，同步显示当前解析式、振幅、周期和相位平移量。
- 支持单参数观察模式：`A`、`omega`、`phi` 或自由组合。
- 支持切换变换顺序：先横向伸缩后平移，或先平移后横向伸缩。
- 支持显示/隐藏基本曲线、中间曲线、关键点和归纳面板。
- 支持拖动或滑块移动当前观察点，并显示当前相位 `omega x + phi`。
- 支持 Applet SDK 基础消息：`applet:ready`、`applet:stateChanged`、`applet:interaction`。

## 资源边界

本资源包含：

- Applet metadata、教师脚本、学生活动任务和草稿审核记录。
- `src/index.html` 可运行原型。
- 参数范围：`-2 <= A <= 2`、`0.5 <= omega <= 3`、`-pi <= phi <= pi`。
- 默认不加入竖直平移 `k`，避免偏离本课时主线。

本资源暂不包含：

- `y = A sin(omega x + phi) + k` 的竖直平移系统讨论。
- 余弦、正切参数函数的完整对照。
- 自动生成诊断题与评分规则。
- 用真实学生作答数据训练反馈策略。

## 文件说明

- `metadata.yaml`：资源元数据、数学范围、交互状态和诊断标签。
- `teacher-script.md`：教师课堂流程、提问链、板书建议和易错处理。
- `student-task.md`：学生预测、观察、记录和反推参数任务。
- `review.md`：草稿阶段审核记录和进入下一状态的门槛。
- `src/index.html`：可独立打开或由平台 iframe 嵌入的 Applet 入口。

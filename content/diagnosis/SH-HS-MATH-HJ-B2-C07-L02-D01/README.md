# 正弦函数性质诊断

资源 ID：`SH-HS-MATH-HJ-B2-C07-L02-D01`

对应课时：`SH-HS-MATH-HJ-B2-C07-L02`「正弦函数的性质」

本资源是 7.1.2 课后 6 到 8 分钟即时诊断，用于判断学生是否真正能用图像比较正弦函数的周期、单调区间、对称元素和关键点一般式。它不作为普通练习题库，而是服务教师快速识别错因并决定回讲路径。

## 教学痛点

- 学生会背周期和值域，但难以用图像解释单调区间端点。
- 奇偶性、对称轴、对称中心容易混淆。
- 最值点和零点的弧度表达容易出现周期遗漏。

## 认知动作

核心认知动作是「比较」：

- 比较同一关键点在相邻周期中的位置，检查是否遗漏 `2kpi`。
- 比较递增半周期与递减半周期的端点，检查单调方向。
- 比较竖直对称轴与中心对称点，检查几何对象是否混用。
- 比较一个周期内的观察和一般式表达，检查是否能迁移到所有周期。

## 诊断目标

学生完成题组后，教师应能判断：

1. 学生能否写出完整的一般解形式。
2. 学生能否区分单调区间和局部观察区间。
3. 学生能否区分对称轴与对称中心。
4. 学生能否把零点、最大值点和最小值点的表达式分开。

## 错因标签

| 标签 | 含义 |
| --- | --- |
| `period_omission` | 只写一个周期内的区间或点，遗漏周期推广。 |
| `monotonic_endpoint_confusion` | 单调区间端点、方向或相邻半周期混用。 |
| `axis_center_mixup` | 对称轴与对称中心混为同类对象。 |
| `extrema_zero_formula_omission` | 零点、最大值点、最小值点的一般式遗漏或互相误用。 |
| `local_interval_overgeneralization` | 把局部观察直接推广成整段单调性。 |

## 题组结构

- 6 题，建议 8 分钟完成。
- 题型包括单选、多选、简答和错误分析。
- 总分 10 分，掌握阈值为 75%。
- 每题都绑定错因标签，教师端优先展示标签统计，而不是只展示总分。

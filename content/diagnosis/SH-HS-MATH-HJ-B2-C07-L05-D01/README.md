## 资源定位

本资源对应沪教版高中数学必修第二册第 7 章 7.3“函数 `y = A sin(omega x + phi)` 的图像”。它放在 `SH-HS-MATH-HJ-B2-C07-L05-A01` 参数变化 Applet 和 `SH-HS-MATH-HJ-B2-C07-L05-M01` 变换顺序动画之后，用 6 题快速暴露学生从图像反推参数时的错因。

核心诊断链路：

```text
读振幅 |A|
-> 读周期 T
-> 计算 omega = 2pi / T
-> 改写 omega x + phi = omega(x + phi / omega)
-> 用关键点验证 A 的符号和相位
```

## 诊断焦点

- 周期与 `omega` 的换算是否倒置。
- `phi` 是否被误当成实际平移量。
- `x + a` 与 `x - a` 的平移方向是否混淆。
- `A < 0` 时是否能处理关于 `x` 轴的翻折。
- 是否能同时使用振幅、周期、平移和关键点，而不是只看一个参数。

## 课堂用法

- 新授课后：学生完成 Applet 的单参数观察后，用本题组做 6 到 8 分钟即时诊断。
- 讲评前测：教师先看错因标签分布，再决定回到 Applet、Manim 动画或板书推导。
- 分层订正：学生订正时必须写出至少一个参数读数理由，不能只改答案。

# 分镜：图像变换顺序解释

## 核心叙事

```text
不要直接记 A、omega、phi 的结论
-> 先把括号内输入拆开
-> omega 改变周期
-> phi 与 omega 共同给出平移量
-> A 改变振幅和上下方向
-> 回到图像读数
```

样例统一使用：

```text
y = -1.5 sin(2x + pi/2)
  = -1.5 sin(2(x + pi/4))
```

## 第 1 幕：先拆输入

画面出现基准曲线 `y = sin x` 和目标解析式 `y = -1.5 sin(2x + pi/2)`。

教师可说：

> 这一节最容易错的地方，是把 `pi/2` 直接看成平移量。我们先不急着画，先拆括号。

数学焦点：

- `omega x + phi` 整体作用在输入 `x` 上。
- `A` 作用在输出 `y` 上。

## 第 2 幕：改写括号

公式由：

```text
2x + pi/2
```

变为：

```text
2(x + pi/4)
```

暂停问题：

> 为什么这里对应的是向左移动 `pi/4`，而不是向左移动 `pi/2`？

期望回答：

- 实际看的是 `x + phi / omega`。
- 本例 `phi / omega = (pi/2) / 2 = pi/4`。
- 正号表示把 `x` 换成 `x + pi/4`，图像向左移动 `pi/4`。

## 第 3 幕：横向压缩

把 `y = sin x` 变为 `y = sin 2x`。关键点之间的横向距离缩短，一周期从 `2pi` 变成 `pi`。

教师追问：

- 周期为什么除以 2？
- 哪些关键点的顺序没有变，只是横坐标变了？

## 第 4 幕：相位平移

把 `y = sin 2x` 变为 `y = sin(2x + pi/2)`，也就是整体向左移动 `pi/4`。

画面保留平移前后的同一关键点，用虚线箭头标注 `left pi/4`。

暂停问题：

> 如果学生说“括号里加 `pi/2`，所以左移 `pi/2`”，你会用图像上的哪个距离反驳？

期望回答：

- 可以比较周期已经变成 `pi` 后的四分之一周期。
- `pi/4` 正好是压缩后一个四分之一周期，而不是原图像的 `pi/2`。

## 第 5 幕：纵向放大和翻折

把 `y = sin(2x + pi/2)` 变为 `y = -1.5 sin(2x + pi/2)`。

数学焦点：

- `|A| = 1.5` 给出振幅。
- `A < 0` 表示关于 `x` 轴翻折。
- 这一步不改变周期和平移量。

暂停问题：

> `A` 的负号改变的是横向位置，还是纵向方向？

## 第 6 幕：读回参数

画面右侧出现读数面板：

```text
amplitude = 1.5
period = pi
shift = left pi/4
```

教师收束：

> 看图像时不要三个参数混在一起猜。先读振幅，再由周期读 `omega`，最后用一个关键点确定相位平移。

## 与 Applet 的衔接

动画结束后切到 `SH-HS-MATH-HJ-B2-C07-L05-A01`：

- 锁定 `A`，只调 `omega`，观察周期。
- 锁定 `omega`，只调 `phi`，观察平移方向和距离。
- 固定 `omega = 2, phi = pi/2`，把 `A` 从正数改成负数，观察纵向翻折。

# 分镜：正弦曲线的来源

## 核心叙事

```text
角 x 在单位圆上转动
-> P 的纵坐标给出 sin x
-> 把 x 作为函数图像横坐标
-> Q = (x, sin x) 留下连续轨迹
-> 一周后终边重合，周期重复有了来源
```

## 第 1 幕：回到单位圆

画面只出现单位圆、坐标轴、角 `x`、终边和动点 `P`。

教师可说：

> 先不要看曲线。我们只问一个问题：单位圆上的点 P 转到这里时，`sin x` 是什么？

数学焦点：

- `P = (cos x, sin x)`。
- `sin x` 来自 P 的纵坐标。

## 第 2 幕：抽取高度

从 `P` 引出水平或竖直辅助线，突出当前高度 `sin x`。动点缓慢旋转到 `0`、`pi/2`、`pi`、`3pi/2`、`2pi` 附近时短暂停顿。

教师追问：

- `pi/2` 时高度为什么是 1？
- `3pi/2` 时高度为什么是 -1？
- 过 `pi` 后正负号发生了什么变化？

## 第 3 幕：放到函数图像

右侧出现函数图像坐标系。把当前角的弧度量放到横轴上，把刚才的高度放到纵轴上，生成点：

```text
Q = (x, sin x)
```

暂停问题：

> Q 的横坐标为什么是 `x`，不是单位圆上 P 的横坐标 `cos x`？

期望回答：

- 函数图像记录的是“输入 x 与输出 sin x”的对应。
- 单位圆中的 `cos x` 是 P 的横坐标，不是正弦函数图像的输入轴。

## 第 4 幕：描出一周轨迹

让 `x` 从 `0` 连续变化到 `2pi`，点 `Q` 留下轨迹。关键角处亮起短标记，最后显示 `y = sin x`。

数学焦点：

- 曲线不是凭空画出的，是 `Q = (x, sin x)` 的运动轨迹。
- 上升、下降、过零、最值都来自单位圆纵坐标的变化。

## 第 5 幕：提示周期重复

动点完成一周后回到同一终边位置。画面提示 `x` 与 `x + 2pi` 的终边重合，因此纵坐标相同：

```text
sin(x + 2pi) = sin x
```

暂停问题：

> 如果继续向右画，为什么会出现同样的波形？

期望回答：

- 终边每转一周重合。
- 单位圆上点的纵坐标重复。
- 图像上的函数值也随之重复。

## 与 Applet 的衔接

动画结束后切到 `SH-HS-MATH-HJ-B2-C07-L01-A01`，让学生亲手拖动 `P`，在关键角处预测 `Q` 的位置，再显示轨迹验证。

本目录用于存放 Manim 数学动画资源。Manim 不替代交互课件，主要负责概念导入、连续推演、证明结构和公式变形叙事。

## 适合 Manim 的内容

- 割线逼近切线。
- 单位圆生成正弦曲线的连续过程。
- 基本不等式几何证明。
- 三角恒等变换的结构化推演。
- 圆锥曲线定义到图像的生成。
- 立体几何截面形成过程。

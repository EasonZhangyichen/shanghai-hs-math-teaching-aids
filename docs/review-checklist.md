# 审核清单

## 适用范围

本清单适用于课程图谱、HTML Applet、Manim 动画、教师脚本、学生活动和平台代码。

## 数学审校

- 定义、定理、符号、公式书写准确。
- 定义域、值域、参数范围、边界条件没有遗漏。
- 极端参数不会产生误导性图像或错误结论。
- 特殊值和关键点标注正确。
- 图像、表格、公式和文字说明相互一致。

## 教学审校

- 明确对应沪教版册别、章节、课时和知识点。
- 明确先修知识、后续知识和常见误区。
- 数字化形式击中真实教学痛点。
- 课堂使用链路清楚：导入、探究、归纳、练习或诊断。
- 有教师提问点、暂停点、板书建议或学生活动任务。

## 交互与动画审校

- 一个资源只聚焦一个主要认知动作。
- 交互动作和数学结构一致。
- 初始信息量克制，结论分步出现。
- 多表征同步联动，而不是并排静置。
- 反馈解释数学规律，不只显示对错。
- 大屏、触控板和普通浏览器都能自然使用。

## Manim 审校

- 动画时长服务课堂节奏，优先 30 到 90 秒。
- 场景转换有数学叙事逻辑。
- 公式、图形、颜色和旁白脚本保持一致。
- 不把 Manim 用于更适合交互探索的知识点。

## 版权与合规

- 不直接复制教材正文、官方课件、教案、视频或商业资源。
- 外部开源项目只借鉴公开思想和结构，代码使用前必须核查 license。
- 引用外部事实、项目或平台时记录来源。

## 版本状态

资源只能按以下状态推进：

```text
draft -> math_review -> pedagogy_review -> classroom_trial -> stable -> deprecated
```

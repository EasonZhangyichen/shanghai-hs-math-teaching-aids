# ADR 0001：采用课时知识图谱 + 多引擎微课件平台

## 状态

Accepted

## 日期

2026-04-30

## 背景

项目目标是构建面向沪教版高中数学教师的数字化课件教具平台。已有研究文档提出了三类方向：

1. 静态 HTML 可视化工具库。
2. 课时知识图谱 + 多引擎微课件平台。
3. AI 生成 + UGC + 审核生态一体化平台。

同时，外部参考项目 `edu-ai-builders/math-viz-kit` 已证明单文件 HTML 数学可视化工具集可快速覆盖大量知识点，但它并不绑定沪教版课时体系，也不包含教师课堂工作流、审校体系和长期版本治理。

## 决策

采用路线 2：课时知识图谱 + 多引擎微课件平台。

平台以沪教版高中数学的册别、章节、课时和知识点为主索引。每个课时聚合 Applet、Manim、教师脚本、学生活动和诊断反馈。交互课件可由 JSXGraph、GeoGebra、CindyJS、D3、Three.js 等引擎支持；不适合交互但适合叙事推演的内容使用 Manim。

## 后果

正面影响：

- 首期可以用三角函数样板包快速落地。
- 后续可以逐步扩展到函数、几何、概率统计、导数和建模。
- 资源天然可版本化、可审核、可复用。
- 平台不会陷入“泛学段工具合集”的定位模糊。

代价：

- 需要先投入课程图谱和内容规范建设。
- 每个课件的研发不仅是前端开发，还包含数学审校和教学设计。
- 需要维护多个课件引擎的适配边界。

## 近期执行

第一阶段只做必修第二册第 7 章“三角函数”样板包，不追求全册覆盖。

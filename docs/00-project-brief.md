# 项目总览

## 定位

本项目是一个面向上海高中数学教师的沪教版课时级数字教具云平台。它不是 PPT 仓库、教案仓库或普通资源下载站，而是以沪教版高中数学课程结构为骨架，聚合可交互 HTML 数字教具、Manim 数学动画、教师课堂脚本、学生活动任务和课堂诊断反馈的教学工作台。

## 推荐路线

采用路线 2：课时知识图谱 + 多引擎微课件平台。

该路线保留 MVP 可落地性，同时为后续 AI 生成、教师共创、教研审核、跨平台嵌入和离线导出留下扩展空间。

## 核心链路

```text
教材册别
-> 章节
-> 课时
-> 知识点
-> 先修/后续关系
-> 教学痛点
-> 数字化切入点
-> Applet / Manim / 诊断任务
-> 教师课堂脚本
-> 学生操作与反馈
```

## 资源包模型

每个课时应沉淀为一个资源包：

- Knowledge Card：教材位置、先修知识、后续知识、核心素养、易错点。
- Interactive Applet：可拖拽、调参、模拟或多表征联动的 HTML 课件。
- Manim Clip：适合连续推演、证明过程或概念生成的数学动画。
- Teacher Script：课堂提问链、暂停点、操作顺序和板书建议。
- Student Task：观察、猜想、验证、归纳、提交。
- Diagnosis：即时练习、错因标签、补救资源推荐。

## 首期 MVP

首期聚焦必修第二册第 7 章“三角函数”，先完成一个能证明平台价值的样板包：

1. 单位圆到正弦曲线。
2. 正弦函数图像与性质。
3. `y = A sin(omega x + phi) + k` 参数变化实验室。
4. 三角函数图像变换诊断任务。
5. 正弦函数来源 Manim 导入动画。
6. 图像变换顺序 Manim 解释动画。

## 不做什么

- 不直接搬运官方平台、教材、PPT、教案或商业平台资源。
- 不把平台做成小初高通用工具集。
- 不为每个知识点强行做复杂动画。
- 不让 AI 生成内容绕过数学审校和教研审核。
